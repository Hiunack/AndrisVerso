🔹 ¿Qué es SQL?

SQL es un lenguaje que usamos para hablar con las bases de datos. Es como darle instrucciones a una computadora para ver, cambiar o contar información.

🔍 Ver datos

Para ver toda la información:

SELECT * FROM nombre_tabla;

👉 Muestra todos los datos de la tabla

Para ver solo algunos campos:

SELECT campo1, campo2 FROM tabla;

👉 Solo muestra columnas específicas

🔢 Contar datos

SELECT COUNT(*) FROM nombre_tabla;

👉 Cuenta cuántos registros hay

SELECT COUNT(*) AS total_solteros FROM nombre_tabla WHERE estado_civil IN ('Soltero', 'soltero', 'SOLTERO');

👉 Cuenta cuántos cumplen una condición

🎯 Filtrar datos

SELECT * FROM nombre_tabla WHERE edad BETWEEN 18 AND 25;

👉 Muestra personas entre 18 y 25 años

💻 Consulta SQL

SELECT id, nombre, cargo, sueldo
FROM empleados
WHERE sueldo > (
    SELECT AVG(sueldo) * 2
    FROM empleados
);

⚠️ Esta consulta muestra empleados cuyo sueldo es más del doble del promedio, lo cual podría indicar una anomalía o posible fraude.

🧠 Explicación de la Consulta SQL

SELECT id, nombre, cargo, sueldo
FROM empleados
WHERE sueldo > (
    SELECT AVG(sueldo) * 2
    FROM empleados
);

🔍 1. SELECT (¿qué quieres ver?)

Indica las columnas que se mostrarán en el resultado:
id, nombre, cargo y sueldo.

📦 2. FROM (¿de dónde salen los datos?)

Especifica la tabla que contiene la información:
empleados.

⚖️ 3. WHERE (condición)

Filtra los resultados:
Solo se mostrarán empleados cuyo sueldo sea mayor que un valor calculado.

🧠 4. Subconsulta

SELECT AVG(sueldo) * 2
FROM empleados

Esta consulta interna realiza dos acciones:
✔ Calcula el promedio de los sueldos (AVG)
✔ Multiplica ese promedio por 2

🔗 5. ¿Cómo funciona todo junto?

Primero se calcula el promedio de los sueldos y se multiplica por 2. Luego, la consulta principal muestra solo los empleados que ganan más que ese valor.

🚨 6. Resultado

Se obtienen los empleados con sueldos desproporcionados, lo que puede indicar una posible anomalía o fraude en la nómina.
