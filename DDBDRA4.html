Backup y Alta Disponibilidad 🚀

¿Qué es un BACKUP?

Un backup es una copia de seguridad de una base de datos que permite recuperar la información en caso de pérdida, error o fallo del sistema.

Importancia

Los backups son esenciales para proteger la información, evitar pérdidas de datos y garantizar la continuidad del sistema.

Pasos para realizar un backup en phpMyAdmin

Exportar:

  • Seleccionar la base de datos
  • Ir a la opción "Exportar"
  • Elegir formato SQL
  • Descargar archivo

Importar:

  • Crear o seleccionar base de datos
  • Ir a la opción "Importar"
  • Seleccionar archivo .sql
  • Ejecutar

Eliminar en SQL

Eliminar campo:

ALTER TABLE empleados DROP COLUMN telefono;

Eliminar tabla:

DROP TABLE empleados;

Eliminar base de datos:

DROP DATABASE empresa;

Consultas Multitablas

JOIN:

SELECT clientes.nombre, pedidos.total FROM clientes INNER JOIN pedidos ON clientes.id = pedidos.cliente_id;

UNION:

SELECT nombre FROM empleados_activos UNION SELECT nombre FROM empleados_inactivos;

Alta Disponibilidad

La alta disponibilidad es la capacidad de un sistema de mantenerse funcionando el mayor tiempo posible, evitando interrupciones.

Estrategias

Replicación: copia de datos en otro servidor en tiempo real.

Clustering: varios servidores trabajando como uno solo.

Mirroring: copia exacta de un servidor (más común en SQL Server).

Diferencias

La replicación sincroniza datos, el clustering distribuye carga y el mirroring mantiene una copia exacta para recuperación inmediata.
