DISEÑO DE BASES DE DATOS - RA3

Aprende a manejar datos como un astronauta digital 🚀

✏️ Modificar datos (UPDATE)

Estructura básica:

UPDATE tabla SET campo = 'valor' WHERE condicion;

Ejemplo simple:

UPDATE tabla SET campo = 'nuevo_valor' WHERE id = 1;

Varios campos:

UPDATE tabla SET campo1 = 'valor1', campo2 = 'valor2' WHERE id = 2;

Condición:

UPDATE tabla SET campo = 'Aprobado' WHERE campo_numerico >= 70;

⚠️ Sin WHERE (cambia todo):

UPDATE tabla SET campo = 'valor_general';

Incrementar valor:

UPDATE tabla SET campo_numerico = campo_numerico + 1 WHERE id = 3;

🧠 Recuerda

UPDATE = Qué cambiar

SET = Nuevo valor

WHERE = A quién cambiar

🛠️ Modificar tablas (ALTER TABLE)

Estructura básica:

ALTER TABLE tabla ACCION;

Agregar una columna:

ALTER TABLE empleados ADD telefono VARCHAR(15);

Modificar tipo de dato:

ALTER TABLE empleados ALTER COLUMN sueldo DECIMAL(10,2);

Renombrar columna (MySQL):

ALTER TABLE empleados CHANGE nombre nombre_completo VARCHAR(100);

Eliminar columna:

ALTER TABLE empleados DROP COLUMN telefono;

Agregar restricción (PRIMARY KEY):

ALTER TABLE empleados ADD PRIMARY KEY (id);

⚠️ Cuidado: estos cambios afectan la estructura de la tabla.

🗄️ Modificar bases de datos

Crear base de datos:

CREATE DATABASE empresa;

Usar base de datos:

USE empresa;

Renombrar base de datos (según gestor):

ALTER DATABASE empresa MODIFY NAME = nueva_empresa;

Cambiar configuración (ejemplo collation):

ALTER DATABASE empresa COLLATE utf8mb4_general_ci;

Eliminar base de datos:

DROP DATABASE empresa;

⚠️ IMPORTANTE: eliminar una base de datos borra toda la información.

✏️ Caso: Corrección de Datos en la Nómina

Durante la auditoría de la empresa TechSolutions S.R.L., se detectó que uno de los empleados tiene su nombre mal escrito en la base de datos. Es necesario corregirlo para mantener la información actualizada.

📊 Registro con error

ID Nombre Cargo Sueldo
8 Laura Mendez Supervisor 47,000

⚠️ El nombre correcto debe ser: Laura Méndez

💻 Consulta SQL

UPDATE empleados
SET nombre = 'Laura Méndez'
WHERE id = 8;

🧠 Explicación

UPDATE empleados → Indica la tabla que se va a modificar.

SET nombre = 'Laura Méndez' → Nuevo valor correcto.

WHERE id = 8 → Se asegura de cambiar solo ese empleado.

🚨 Sin la condición WHERE, se cambiarían todos los nombres de la tabla.
